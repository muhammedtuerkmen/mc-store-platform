# Contributing

Thanks for your interest in contributing. This project is open source and community driven. The goal is to build a high‑quality Minecraft store platform that is transparent, flexible, and fair.

## Ways to contribute

- Report bugs and edge cases
- Propose features and improvements
- Improve UI/UX and accessibility
- Add documentation and examples

## Development setup

```bash
npm install
npm run dev
```

## Guidelines

- Keep changes small and focused.
- Prefer reusable components and config‑driven data.
- Match existing code style and formatting.
- Update documentation when behavior changes.
- Include screenshots for UI changes.

## Submitting a Pull Request

1. Fork and create a branch: `feature/your-change`
2. Make changes and test locally
3. Open a PR with a clear summary and screenshots if UI is affected

## Code of Conduct

By contributing, you agree to follow the Code of Conduct.
